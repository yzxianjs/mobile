# demo01
## 下载项目
git clone git@github.com:yzxianjs/mobile.git

## 项目依赖安装
```
yarn install
```

### 编译运行项目
```
yarn run serve
```

### 项目打包
```
yarn run build
```

### 测试
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
